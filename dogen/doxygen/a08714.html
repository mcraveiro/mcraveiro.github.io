entities::comment process_comment ( const std::string &  c)
static

Processes comments in dia diagrams.

Comments can be stored in class objects, member functions, note objects, etc.

Any line starting with the well defined marker #DOGEN - known as the instruction marker - will be interpreted as carrying an instruction in a key-value pair form, where the key and the value are separated by the assignment operator =.
