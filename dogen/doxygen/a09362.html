boost::optional< entities::logical_name > build_containing_element_name ( const entities::logical_name &  n) const

Creates the name of the containing element, derived from the element's name.

A containing name is the name of an element that is able to contain other elements. This is typically a module, but not always.

If the supplying name does not represent a valid containment, returns null. At present there are two cases of "invalid" containment: the global module and the model's module.
