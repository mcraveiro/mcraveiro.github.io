◆ binding_point()

const boost::optional< dogen::variability::entities::binding_point > & binding_point ( ) const

Override binding point for this feature.

If the default binding point is supplied for a bundle, the features cannot supply individual binding points. Conversely, if not supplied, they must supply their individual binding points.

◆ requires_optionality()

bool requires_optionality ( ) const

If true, the feature's static configuration will have an optional type.

This is only required if the feature template is optional and has no default value.
