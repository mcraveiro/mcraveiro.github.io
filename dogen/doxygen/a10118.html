Detailed Description

Context for all logical transformations.

Contains all of the external data required for the transformations to execute. It's not ideal to have a huge "global" class, with lots of unrelated state; however, over time, we found that a number of arguments were being supplied across the call graph, resulting in a lot of repetitive code. The context gathers together all of these.
