Describes the geometry of physical space.

const std::unordered_map< std::string, std::vector< std::string > > & template_instantiation_domains ( ) const

Provides all of the domains to be used for template instantiation.

Example of a domain is "masd.facet" which contains the list of all available facets. Templates (facet and profile) are then instantiated over this range, depending on user choices.
