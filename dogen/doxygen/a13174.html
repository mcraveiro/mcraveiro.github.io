Detailed Description

The exception thrown when an enum has an unexpected value.

This exception should be thrown only when:

  • the current value of an enumeration is not part of its domain, .e.g. an int was cast into an enumeration with a value which is not valid; or
  • a given valid value on an enumeration is not supported in a given context.
